# Breast Cancer Classification Using Deep Learning

This project implements a CNN–BiLSTM–Attention based deep learning model for breast cancer classification using histopathology images.

## Features
- Deep learning based medical image classification
- CNN with MobileNetV2 feature extraction
- BiLSTM with Attention mechanism
- Streamlit web application for real-time prediction
